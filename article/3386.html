微软正着手调查 Outlook 搜索无法在 Windows 11 桌面应用中显示近期邮件的问题。由于 Outlook 的搜索功能使用本地 Windows 搜索服务来索引电子邮件，因此该问题会影响 POP、IMAP 和脱机 Exchange 帐户。而 Microsoft 365 和联网 Exchange 账号不受影响。

微软解释道：“在 Windows 11 上的 Outlook 桌面中搜索时，您可能不会在搜索结果中看到最新的电子邮件。如果您在控制面板中检查 Windows 索引选项，索引可能运行缓慢或可能已暂停”。

临时解决方案

该公司目前正在努力解决这个已知问题，并为受影响的客户共享一个临时修复程序，直到有解决方案可用。微软建议 Windows 11 用户通过禁用 Windows 桌面搜索并切换到 Outlook 的内置搜索引擎作为解决方法来缓解 Outlook 搜索错误。

要禁用 Outlook 的 Windows 桌面搜索服务，您必须使用此处概述的步骤添加新的 PreventIndexingOutlook 注册表项。

这个问题并非首次

这是微软第二次提醒 Outlook 用户存在电子邮件搜索问题。去年 6 月开始升级到 Windows 11 后，Outlook for Microsoft 365、Outlook 2019 和 Outlook 2016 中遇到了搜索问题。

正如受影响的人所说，在 IMAP/POP 帐户上搜索特定电子邮件时，Outlook 不再显示结果，在某些情况下会导致应用程序完全冻结。这些问题在 2021 年 6 月开始出现，就在第一个 Windows 11 预览版本浮出水面之后，影响了 POP 和 IMAP 帐户在 PST 或 OST 文件中本地存储的电子邮件和其他项目。
