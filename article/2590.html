近日在中国深圳召开的“NAS Workshop”上，泄漏了英特尔第 13 代 Raptor Lake 桌面 CPU 平台的详细信息。这张幻灯片让我们很好地了解了即将推出的处理器应该具备的特性，但它也缺少一个关键特性-- PCIe Gen 5 M.2。

根据幻灯片所展示的信息，英特尔的第 13 代 Raptor Lake CPU 看起来是第 12 代 Alder Lake CPU 的优化版本。 它们将基于相同的 10nm ESF“Intel 7”工艺节点，采用相同的 P-Core 和 E-Core 设计，并支持现有主板。

不过 Raptor Lake 在细节方面又有所调整，首先我们来看看它的规格

● 最多 24 个核心和 32 个线程

● 全新的 Raptor Cove CPU 核心（更高的 P-Core IPC）

● 基于 10nm ESF 'Intel 7'工艺节点

● 支持现有 LGA 1700 主板

● 支持双通道 DDR5-5600 内存

● 20 个 PCIe Gen 5 通道

● 增强超频功能

● 125W PL1 TDP (Flagship SKUs)

Raptor Lake 依然使用 LGA 1700 平台。Chipzilla 确认 Raptor Lake CPU 将与现有的基于 600 系列芯片组的 LGA 1700 主板兼容。但与每一代一样，主板制造商将提供基于 700 系列芯片组的全新主板阵容，这些芯片组将配备更高的 I/O 通道。

除此之外，Raptor Lake 芯片将支持 DDR5-5600 速度，与 Alder Lake 支持的原生 DDR5-5200 速度相比，这是一个很好的提升。 CPU 还将继续在 D4 系列主板上支持 DDR4-3200。

这为当前运行主流 Core i3 或 Core i5 CPU 并希望升级到更高端芯片的用户提供了一个很好的升级途径。他们可以简单地用更高端的 Core i7 或 Core i9 SKU 替换现有的第 12 代 CPU，这将提高他们 PC 的整体性能。
