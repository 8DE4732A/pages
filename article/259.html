以“为玩家打造最佳浏览器”为宣传口号，在近日上线的 Edge 103 稳定版中针对游戏进行了“重大”优化。这些优化可以参考本站此前发布的文章，除了游戏改进外新版本也包含一些更改、安全性升级和小幅改进。

根据官方文档中的更新日志，Edge 103 带来了以下功能更新：

● 能够控制自动账户切换

GuidedSwitchEnabled 策略允许 Microsoft Edge 在 Microsoft Edge 检测到链接是个人或工作链接时提示用户切换到适当的账号。

● 客户端证书切换器

此功能将为用户提供一种在访问需要 http 证书身份验证的站点时清除记住的证书并重新显示证书选择器的方法。无需手动退出 Microsoft Edge 即可完成切换。

● 更可靠的网络防御

借助针对 Windows 上的 Microsoft Edge 重写的 Microsoft Defender SmartScreen 库，以更可靠的保护浏览 Web。 NewSmartScreenLibraryEnabled 策略将允许企业客户继续使用旧版本的库，直到它在 Microsoft Edge 版本 105 中被弃用。

● Microsoft Edge 地址栏中的工作搜索横幅

此横幅通过将搜索焦点缩小到仅限工作的结果来帮助您保持工作流程。要查看您的组织以工作为重点的结果，请在搜索开始时选择横幅。要定向到您组织的工作场所搜索结果页面，请在搜索的任何位置选择横幅。使用 AddressBarMicrosoftSearchInBingProviderEnabled 策略打开或关闭此功能。

还有一些政策变化和更新：

● GuidedSwitchEnabled - 已启用引导开关

● InternetExplorerZoomDisplay - 在 IE 模式选项卡中显示缩放，其中包含 DPI 比例，就像在 Internet Explorer 中一样

● LiveCaptionsAllowed - 允许实时字幕

● OriginAgentClusterDefaultEnabled - 默认启用 Origin-keyed 代理集群

Microsoft Edge 将自动更新，但您可以通过导航到 edge://settings/help 来强制安装 Edge 103。
