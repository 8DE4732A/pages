通过定制的水循环装置，极限超频爱好者 SkatterBencher 对 AMD 尚未发布的 Ryzen Threadripper 5990X HEDT CPU 进行了超频，所有 64 个核心的时钟频率都达到了惊人的 4.82GHz。

本次 SkatterBencher 测试的 AMD Ryzen Threadripper CPU 是 5990X，目前官方并未正式发布。他获得的是 AMD 的工程样片，内置 64 个核心、128 个线程，三级缓存容量达到了 288MB，时钟频率为 2.65GHz（平均为 2.27 GHz）。

CPU 在配备 32 GB (8 x 4) DDR4-2144 内存的 ASUS ROG ZENITH II Extreme Alpha 上进行了测试。在散热方面使用了EK-Quantum 定制回路水冷却器。

此工程样片的 OPN 代码为“100-000000443-40_Y”，类似于显示在各种基准数据库中的 AMD Ryzen Threadripper 5000 样本。这证实了 AMD 确实有标准的 Non-Pro 芯片作为工程样品但出于未知原因决定不发布它们的事实。

根据 AMD 最近的 HEDT 战略，该公司似乎将主要专注于工作站和 Pro 领域，这意味着在可预见的未来，标准的 Threadripper X 系列芯片将在行动中消失。另一件重要的事情是 CPU 可以与 TRX40 ZENITH II Extreme 配合使用，而 Threadripper 5000WX CPU 并不适用于除了价格更高的 WRX80 平台之外的任何东西。

SkatterBencher 以擅长超频享誉业内。 AMD Ryzen Threadripper 5990X CPU 的最佳结果是通过手动超频实现的，这并不令人意外。唯一的缺点是最大单线程提升频率有限，在 Ryzen Threadripper 5990X 芯片上的 8 个 CCD 中，CCD4 提供了最好的调谐结果，在 1.45V 下达到了 4.82 GHz。

性能方面，AMD Ryzen Threadripper 5990X CPU 手动超频后在 AI Benchmark 中获得超过 50% 的提升，在 Cinebench R23 中获得超过 58% 的提升，在 3DMark CPU Profile 基准测试中性能提升超过 80%。 CPU 还打破了 Cinebench R23 多线程中的 100K (100,000) 分数障碍，这简直太疯狂了。在 Cinebench 运行期间，该芯片最高时钟速度为 4.525 GHz，CPU 温度高达 95C。 CPU 在手动超频时消耗了疯狂的 691W 封装功率。

不仅如此，SkatterBencher 还提供了在 AMD Ryzen Threadripper 5990X CPU 上在 5 分钟内达到 4.65 GHz 的指南。这是否会以与 5995WX 相同的方式工作，在用户可以在 DIY 领域购买芯片之前，还有待观察。 MSI WS WRX80、ASUS WRX80 PRO WS SAGE SE、ASRock WRX80 Creator 和 Gigabyte WRX80 SU8 等某些主板将在 Threadripper Pro CPU 上提供超频支持。
