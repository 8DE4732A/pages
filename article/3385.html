自去年 10 月发布以来，Windows 11 系统经历了多次版本更迭，在近期 Puget Systems 展开的和 Windows 10 对比评测中，发现 Windows 11 与上次相比有了显着改进。今天早些时候，国外科技媒体 Phoronix 将 Windows 11 与 Ubuntu、Clear Linux 进行对比，看看微软操作系统的表现如何。

测试的操作系统

● Windows 11 Pro

● Ubuntu 22.04 LTS

● Ubuntu 22.04 LTS with Linux 5.18 stable kernel

● Intel's Clear Linux 36580

测试平台为

● Intel Core i9 12900K

● ASUS ROG STRIX Z690-E GAMING WiFi motherboard

● 2x 16 GB DDR5-6000 memory

● 500GB WD_BLACK SN850 NVMe SSD

● Radeon RX 6800 XT GPU

与 Puget Systems 发现的不同，Phoronix 测试表明 Windows 11 在部分选项上优于 Linux 和 Ubuntu。事实上，在整体平均性能方面，Windows 11 已经落后于经过测试的 Linux 发行版，这与上次观察到的情况形成鲜明对比，上次观察到 Windows 11 在许多情况下都明显更快。
