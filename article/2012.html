面向 Dev 频道的 Windows Insider 项目成员，今天发布了服务渠道更新 KB5016322，用户安装后版本号为 Build 25151.1010。新版本并未引入任何新的东西，只是为了测试服务管道。

在完整公告中写道：“我们开始推出累积更新内部版本 25151.1010 (KB5016322)。此更新不包含任何新内容，旨在测试我们在 Dev 频道中构建的服务管道”。因此，周三发布到开发频道的 build 25151 中宣布的所有错误修复和改进都将继续保持。

有趣的是，这些用于服务管道测试的更新虽然没有引入任何新的功能，但可能会导致某些服务无法正常运行。例如 Windows 10 Build 21292.1010 导致 ARM PC 上的 x64 仿真出现问题。
