Beta 频道 Edge 浏览器近日获得版本更新。对于普通用户来说，Edge 104 Beta 虽然没有新的功能，但是对于企业用户来说引入了诸多新的策略，方便 IT 管理员进行精细化管理。

新政策：

● AllowedDomainsForApps - 定义允许访问 Google Workspace 的域

● AskBeforeCloseEnabled - 在关闭具有多个选项卡的浏览器窗口之前获取用户确认

● BrowserCodeIntegritySetting - 配置浏览器进程代码完整性保护设置

● DoubleClickCloseTabEnabled - 启用 Microsoft Edge 中的双击功能（仅在中国可用）

● ImportOnEachLaunch - 允许在每次 Microsoft Edge 启动时从其他浏览器导入数据

● QuickSearchShowMiniMenu - 启用 Microsoft Edge 迷你菜单

● PasswordManagerRestrictLengthEnabled - 限制可以保存在密码管理器中的密码长度

● PDFXFAEnabled - 在原生 PDF 阅读器中启用 XFA 支持

● TextPredictionEnabled - 默认启用文本预测

过时的策略：

● U2fSecurityKeyApiEnabled - 允许使用已弃用的 U2F 安全密钥 API

Edge Beta 适用于 Windows、macOS、Linux 和 Android。您可以从桌面平台的官方网站和 Android 的 Google Play Store 下载浏览器。 Microsoft 计划在 2022 年 8 月 4 日这一周发布 Edge 104 稳定版。
