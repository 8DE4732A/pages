根据 Valve 公布的最新一期 Steam 软硬件报告，Linux 在玩家系统中的占比达到了 1.18%，相比较上月增加了 0.06 个百分点。macOS 占比达到了 2.45%，相比较上月增加了 0.25 个百分点。微软 Windows 系统占比达到了 96.37%，相比较上月回落了 0.31 个百分点。

去年 6 月，Linux 在玩家系统中的占比仅为 0.9%，因此在过去一年时间里增长速度很大。这很大程度上归功于 Steam Deck 的出货，让更多用户通过 SteamOS 3 的方式进行 Linux 游戏。

在 Linux 细分上，“AMD Custom GPU 0405”GPU 增加了 2.37%，这是 Steam Deck 的图形处理器。Steam Deck 的 GPU 为 Linux 游戏系统上的所有Steam提供动力，大约占7.57%。这现在超过了 Radeon RX 480"Polaris"和 GeForce GTX 1060，成为 Linux 上最受欢迎的GPU。

同时在CPU方面，Valve 报告说英特尔的 Steam on Linux 游戏 CPU 市场份额增加了 0.68 个百分点，达到 54.4%，AMD 为 45%。因此，这似乎表明，除了新的Steam Deck销售之外，Linux 游戏在百分比上仍有一些增长。
