早在今年 3 月，英特尔就宣布了 Arc Alchemist 移动 GPU 产品线。但在今夏登陆美国市场之前，蓝厂还是选择了在中国市场的部分笔记本电脑上首发。有趣的是，在最新分享的数据中，英特尔自信地喊出了 A730M / A770M 移动显卡，比英伟达 RTX 3050 Ti Mobile / RTX 3060 Mobile 更快的口号。

规格方面，A730M / A770M 采用了台积电 6nm 工艺生产的 ACM-G10 GPU，前者具有 24 个 Xe Core / 24 个光追单元、时钟频率 1100 MHz，后者多了 8 个 Xe Core 和光追单元、且频率高达 1650 MHz 。

显存方面，A730M 为 192-bit @ 12GB GDDR6，而 A770M 则是 256-bit @ 16GB GDDR6 —— 综合性能也是前者的两倍。不过在测试平台方面，英特尔使用了配置不尽相同的四台笔记本电脑。

● Arc A730M 基于预生产机型，搭配了 14C / 20T 的酷睿 i7-12700H 标压移动处理器 + 16GB DDR5-4800 内存。

● RTX 3050 Ti Mobile（60W）基于 ROG Zephyrus M16 笔记本，搭配 8C / 16T 的酷睿 i7-11800H（Tiger Lake）芯片 + 16GB DDR4-3200 内存。

需要指出的是，RTX 3050 Ti Mobile 还有 70W / 80W 等 SKU，因而 ROG Zephyrus M16 的性能数据并不具备太大的参考意义。

此外 Arc A770M 和 RTX 3060 Mobile 机型的对比更加离谱，后者常见 TGP 功耗在 60-115W 之间、但本次对比仅选用了 85W 的版本。

● Arc A770M 基于预生产机型，搭配 14C / 20T 的酷睿 i9-12900HK（Alder Lake-H）旗舰移动 CPU + 16GB DDR5-4800 内存。

● RTX 3060 Mobile（85W）基于 MSI Pulse GL66 笔记本，其余规格与 ROG Zephyrus M16 几乎一致。

截图（via Tom's Hardware）

在加上一堆前置条件之后，英特尔在 1080p 分辨率下得出了 Arc A730M 较 RTX 3050 Ti Mobile 快 13% 的结论。

该公司并未给出 A730M 的 TGP 功耗数据，但它通常在 80-120W 之间。至于 RTX 3050 Ti Mobile，则是 35-80W 之间（这里选用了不高不低的 60W SKU）。

此外英特尔宣称 Arc A770M 较 RTX 3060 Mobile 领先 12%，但同样没有给出前者的 TGP 功率数据（通常在 120-150W 之间），而后者通常在 60-115W（这里选用了折中的 85W）。

总体而言，我们还是很高兴见到英特尔能够具有媲美 RTX 3060 Mobile 的性能。至于产品的竞争力，除了看定价策略，还得看 Arc Alchemist GPU 的一些附加特性，比如率先提供了 AV1 硬件编解码器。

最后，如果搭配 12 代 Alder Lake CPU，Arc Alchemist GPU 还可获得一些特殊加成 —— 比如 Hyper Encode 和 Deep Link 。

遗憾的是，早期的图形驱动兼容性，仍存在较大的改进空间。但愿在赴美上市前，英特尔能够努力完善 Arc Alchemist GPU 的游戏与软件体验。
