有望和 Fedora Workstation 和 Fedora Server 一样，Fedora Core 在 Fedora 37 开发周期中提升为正式发布的“版本”（Edition）。

Fedora CoreOS 是一个自动更新、最小化的操作系统，用于容器工作负载，并将安全性和可扩展性作为关键设计元素。Fedora CoreOS 已经开发了几年，目标是结合CoreOS Container Linux和Fedora Atomic Host的最佳特性。

早在 Fedora 34 开发周期中，Fedora CoreOS 的开发者希望能够晋升为官方状态，但当时并没有成功。反馈意见已经得到解决，希望通过 Fedora 37 周期使 Fedora CoreOS 成为与 Fedora Workstation、Server 和 IoT 构建并列的官方版本。

希望 Fedora CoreOS 成为"官方"版本是为了鼓励Linux容器工作负载对它的采用。将 Fedora CoreOS 提升为官方版本仍然需要由 Fedora 工程和指导委员会（FESCo）来决定。
