随着国行首发 Arc A380，我们对英特尔锐炫桌面独显的跑分和游戏性能都有了初步的了解。预计未来几个月，蓝厂也会努力 Arc Alchemist GPU 推向国际市场。不过对于普通游戏玩家来说，还请在购买前仔细阅读官方发布的系统需求指南。

此前，英特尔已强调必须启用 PCIe Resiable BAR（简称 ReBAR）以获得最佳性能表现。

虽然指南中暗示了 Arc Alchemist GPU 也支持 AMD 平台的智能存取（SAM）功能，但该公司似乎更推荐消费者选用自家十代酷睿及更新的平台。

不过今天，该公司已在致 Neowin 的一份声明中证实，Arc 桌面确实支持 SAM 智能存取功能 —— 意味着 AMD 锐龙平台也能够与英特尔自家板 U“同台竞技”。

正如我们在上月的一篇博客文章中所描述的那样，Arc 图形产品的推出、也考虑到了跨硬件平台的体验，以更好地为广大客户服务。

目前我们正在支持可启用 ReBar 的 Intel 平台，并将随时间推移而添加对可启用 Smart Access Memory 功能的 AMD 平台的支持。

但是无论 AMD 还是 Intel，并非所有老平台都全面支持 ReBar 或 SAM 功能。

比如红队这边需要需要搭配锐龙 3000 系列处理器（不含 3000G APU）和 500 系列 AM4 主板。

此外即将于今年下半年到来的 Zen 4 AM5 CPU 和 600 系列芯片组平台，也有望支持 SAM 。

最后，从近期大家的反馈来看，在 AMD 平台上使用 Arc A380 的优化仍不够到位。

与 Intel 平台相比，Arc Alchemist GPU 的性能会受到 AMD 平台的严重拖累。

我们也只能给英特尔更多的时间，以期蓝厂能够尽快补足这方面的短板。
