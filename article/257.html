通过键盘快捷方式可以提高 Microsoft Edge 浏览器的使用效率，但微软正开发一种新的方法。正如 Reddit 社区网友 Leopeva64-2 和科技媒体 Neowin 上发现的那样，微软正在测试 Edge 的启动器，它可以让你使用快速命令来完成常见任务。

这个新启动器目前已经在 Edge 的 Canary 和 Dev 频道进行测试。感兴趣的用户可以在地址栏中输入“edge://flags/#edge-devtools-toaster”，然后将“DevTools Toaster”设置为启用，您可以通过按键盘上的 Ctrl+Shift+Space 按钮来召唤它。

当你这样做时，你会在 Edge 的网页中间看到一个浮动框，提示你输入键盘。这个盒子看起来非常干净，就像 macOS 上的 Spotlight，它漂浮在屏幕中间你正在处理的东西之上。

然后，您可以输入标签、窗口或清除等关键字，以获取有关关闭标签或清除浏览历史记录等常见任务的建议。你甚至可以看到你的历史，但请注意，命令栏并非支持所有任务，因此您可能仍需要手动转到 Edge 的设置菜单来执行任务。

总体而言，这似乎是一个不错的功能，但由于它正在 Edge Insider 频道中进行测试，因此并非每个人都能在 Edge 的稳定版本中获得此功能。
