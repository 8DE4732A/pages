WCCFTech 刚刚汇总了 AMD Radeon RX 7900 XT 这款“RDNA 3”Navi 31 显卡的规格、性能、定价和上市日期等信息。作为红队的下一代旗舰，其有望将玩家的 PC 游戏体验推升到前所未有的水平。此前该公司已通过 Radeon RX 6000 系列“RDNA 2”显卡，证明了它可与绿厂的 GeForce RTX 产品线一较高下。

（via WCCFTech）

为了给 PC 游戏玩家带来更棒的体验，AMD 正积极通过各种架构和软件创新来提升性能。

除了 Infinity Cache 无限缓存、FidelityFX 超分辨率（FSR）、SAM 智能存取，现代游戏玩家还可尽享光追、DX12 Ultimate 和 Visual Upscalers 等技术。

与此同时，下一代 GPU 正变得更加费电且价格高昂。AMD 预告的 RDNA 3 GPU 特性包括：

● 5nm 工艺节点

● 先进的小芯片封装

● 基于新架构的计算单元（CU）

● 优化的图形管道

● 下一代 Infinity Cache 无限缓存

● 超 50% 的每瓦性能提升（相较于 RDNA 2）

AMD Radeon 技术事业部工程高级副总裁 David Wang 指出，今年晚些时候问世的 Radeon RX 7000 系列显卡，将带来超过 50% 的每瓦性能提升。

此外作为 RDNA 3 的旗舰，Navi 31 GPU 的内部代号为“Plum Bonito”，而 Navi 21 则是“Sienna Cichlid”。

红队在 Radeon RX 6000 系列显卡上首次引入了光追硬件，在落后英伟达一代的情况下，它显然难以与 RTX 30 系列 Ampere 产品线相提并论。

不过随着下半年 Radeon RX 7000 和 GeForce RTX 40 新品的发布，两家公司的竞争势必会变得更加激烈。

除了光追，RDNA 3 GPU 的图形管道优化，也可带来较 RDNA 2 更高的 GPU 频率。

此前 Radeon RX 6000 系列已经接近 3 GHz，而改进的 5nm 工艺节点有望让 Radeon RX 7000 系列冲破 3 GHz 大关。

另一方面，据说绿厂那边的 RTX 40 系列 Ada Lovelace GPU 也会用上更高效的 4N（5nm 优化）工艺节点，以实现高达 3 GHz 的频率。

此外 AMD 将为 RDNA 3 图形架构带来对更多高级特性和丰富的软件生态系统支持，比如 AV1 硬件加速解码和有助于 AI 学习的 WMMA 指令。

为了让玩家更加尽兴，硬件加持的下一代 FSR 3.0 技术有望与英伟达 DLSS 套件一较高下，且 GPU 有望支持全新的 PCIe 5.0 协议 @ 高达 128 GB/s 的传输速率。

在增强 SMART Access Memory 智能存取内存的同时，红队还会进一步发展出 Smart Access Storage 智能访问存储。

设计符合微软的 Direct Storage API，可在游戏中提供更快的加载时间和更好的纹理流，另外还有 DP2.0 + HDMI 2.1 等显示特性。

详细规格方面，AMD 会在下一代换用“WGP”工作集群来表示。每个 WGP 容纳双计算单元（CU），辅以翻倍的 SIMD32 数量（RDNA 2 GPU 上仅为每 CU / 2 个）。

至于确切的代工合作伙伴，目前尚不清楚 AMD 会选择三星还是台积电的 6nm 工艺。

RDNA 3 Navi 31 GPU 预计拥有 48 个 WGP 集群、12 个 SA 和 6 个 SE —— 总计 12288 个流处理器，低于早前的爆料。

为了不拉低整体算力（FP32 @ 75 TFLOPs），AMD 或在其旗舰 SKU 上使用超过 3.0 GHz 的疯狂频率（性能达到 RX 6950 XT 的 2.3 倍）。

此外 Navi 31 GPU 将结合 6 个 MCD，每芯片 64MB 无限缓存 + 双 32bit 内存控制器（384-bit 总线接口）。

为了将各个小芯片高效结合到一起，Navi 31“RDNA 3”MCD 将用上新一代 Infinity Fabric 互连，搭配 256-384 MB 的 Infinity Cache 。

基于此，Navi 31 旗舰显卡或配备 24GB 显存，与英伟达 RTX 3090 / 4090 持平。与此同时，两家公司的显卡功耗也会有所增长。

作为参考，RX 6950 XT 的 TBP 功耗已达 335W 。若预估性能提升超 2 倍，也预期 RX 7900 XT 的 TBP 接近 400-450W 。

在此情况下，公版和非公显卡厂商都可能需要依赖更加高端的散热器 —— 比如双槽设计的 120mm 一体水冷方案。

以下是 Radeon RX 7000 系列 RDNA 3 显卡的预估 FP32 性能参数：

● RX 7900 XT： ~75 TFLOPs（假设 3.0 GHz 时钟）

● RX 6950 XT： 23.80 TFLOPs（2324 MHz 加速频率）

● RX 6900 XT： 23.04 TFLOPs （2250 MHz 加速频率）

● RX 6800 XT： 20.74 TFLOPs（2250 MHz 加速频率）

● RX 6800： 16.17 TFLOPs（2105 MHz 加速频率）

最后推测下 AMD Radeon RX 7000 系列显卡的定价和上市日期，原本我们预估红队会优先推出 Navi 31 / 32 / 33 GPU 。

不过近日有新爆料称，AMD 会先在 10 月下旬 - 11 月中旬（2022 年四季度）发布 Navi 31 / 33 SKU，然后 Navi 32 SKU 会稍晚些到来。

有趣的是，AMD Zen 4 锐龙 7000 系列（以及英特尔 13 代 Raptor Lake）台式 CPU 的上市窗口也与之接近。

价格方面，RX 7900 XT 很可能超过 RX 6950 XT 旗舰 SKU 的 1000 美元定价，同时在 600-800 美元的戏份市场提供更具性价比的选项。
